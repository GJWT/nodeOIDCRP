matchTo(uri, method, req, contentType, accept)

Create the information pieces necessary for sending a request. Depending on whether the request is done using GET or POST the request is placed in different places and serialized into different formats.
Parameters:
Name Type Description
uri string The URL pointing to where the request should be sent
method string Which method that should be used to send the request
req Message The request as a :py:class:`oicmsg.message.Message` instance
contentType string Which content type to use for the body
accept bool Whether an Accept header should be added to the HTTP request
